How Invite Codes Work
1
Generate a code for a specific plaza and unit below. You choose how many times it can be used and when it expires.
2
Share the code with your tenant via WhatsApp, SMS, or in person. The code is short and easy to type — e.g. AH-K7R2
3
Tenant registers at rentms.com/register and enters the code. The system automatically links them to your unit, sets the rent, and creates their lease.
